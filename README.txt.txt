
# Project Title

Generate basic CRM - Contact / Opportunity / Customer.



## Installation

**Client:** 

Install 'crm-frontend' with npm install

```bash
  	npm install crm-frontend
  	cd assessment-crm-frontend
	npm start
```

**Server:**

[JDK 11]
[Spring Boot 2.7.6]
[JUnit 4.12]

```bash
	cd CRM_BASIC
  	mvn spring-boot:run
```



## Tech Stack

**Client:**  : React, JavaScript, Rest API, HTML, CSS, Selenium.

**Server:** : TDD, Unit Testing, Java, Spring Boot, Rest API, MySQL.
